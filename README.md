# 🏎️ JAJAN MAINAN - Toy Store App 🧸

**Nama: Irvani Canlusty Missa**

**Kelas: A**

**Mata Kuliah: Pemograman Aplikasi Bergerak**


JAJAN MAINAN merupakan Sebuah aplikasi katalog toko mainan yang bernama "JAJAN MAINAN". Aplikasi ini dibangun menggunakan Flutter dengan manajemen state Provider untuk pengalaman belanja yang mulus.

---

## ✨ Fitur Utama

- **CrazyGames Dark Aesthetic**: Tema gelap pekat (`#0F0F13`) untuk kenyamanan mata.
- **Super Compact Grid**: Layout 6-kolom yang responsif untuk memaksimalkan visibilitas produk dalam satu layar.
- **Real-time Cart System**: Penambahan barang ke keranjang dengan notifikasi SnackBar instan.
- **Shopping Cart Badge**: Indikator jumlah barang di keranjang yang diperbarui secara otomatis.
