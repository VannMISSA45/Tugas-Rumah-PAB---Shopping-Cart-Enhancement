# 🏎️ JAJAN MAINAN - Toy Store App 🧸

[![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-0175C2?style=for-the-badge&logo=dart&logoColor=white)](https://dart.dev)

Sebuah aplikasi katalog toko mainan yang bernama "JAJAN MAINAN". Aplikasi ini dibangun menggunakan Flutter dengan manajemen state Provider untuk pengalaman belanja yang mulus.

---

## ✨ Fitur Utama

- **CrazyGames Dark Aesthetic**: Tema gelap pekat (`#0F0F13`) untuk kenyamanan mata.
- **Super Compact Grid**: Layout 6-kolom yang responsif untuk memaksimalkan visibilitas produk dalam satu layar.
- **Real-time Cart System**: Penambahan barang ke keranjang dengan notifikasi SnackBar instan.
- **Shopping Cart Badge**: Indikator jumlah barang di keranjang yang diperbarui secara otomatis.
